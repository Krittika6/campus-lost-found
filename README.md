# 🎒 Campus Lost & Found

A full-stack MERN web application that allows students to report lost/found items and securely communicate with item owners.

🌐 **Live Frontend:** https://your-vercel-url.vercel.app  
🖥 **Backend API:** https://your-render-url.onrender.com  

---

## 🚀 Features

- 🔐 JWT-based authentication (Register / Login)
- 📦 Post lost or found items
- 🖼 Upload item images via Cloudinary
- 📄 Pagination for scalable browsing
- 🧑 Owner-only controls (Delete / Mark Returned)
- 💬 In-app messaging between users
- ☁️ Fully deployed (Render + Vercel)

---

## 🛠 Tech Stack

**Frontend**
- React (Vite)
- Axios
- Tailwind CSS

**Backend**
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Cloudinary (image uploads)

**Deployment**
- Render (Backend)
- Vercel (Frontend)
